Acquisitions UI : CPS
========

##Usage
To run this code you will need npm, gulp and node (nodemon or forever is a plus)

To install all npm dependencies

```sh
$ npm install
```

To build the source

```sh
$ gulp
```

To watch for file changes

```sh
$ gulp watch
```

To run unit tests with karma

```sh
$ gulp test
```

To run e2e tests with protractor

```sh
$ gulp e2e
```
